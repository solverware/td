\subsection{Creating an MS Excel Analysis Spreadsheet}
Data from tests are provided as text files having the CSV format, meaning values are provided in rows with columns separated by columns. Such files use the extension CSV. While CSV file are easily imported into Excel, they are effectively dumb spreadsheets as there are no formulas or calculations. As such, any work done by the spreadsheet would entail editing each spreadsheet to add such functionality. This is impractical when there is more than one spreadsheet to analyze as was the case here.

The solution was to create an application (named PerfTest) that reads the CSV file and creates an actual Excel spreadsheet having the XLSX file extention. This design provides an entry point to add Excel built-in functions into the spreadsheet to analyze the data as well as custom functions to provide physical properties.
